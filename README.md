This is a Kotlin-based Android software called SocialSpark to help Cora retain social relationships throughout hectic days by providing simple, time-based social recommendations.

The features of the app:

The time-based Input 

Users input a keyword related to the time of day:
- "Morning"
- "Afternoon"
- "Dinner"
- "Night"
  
<img width="323" height="75" alt="Suggestion" src="https://github.com/user-attachments/assets/adae3d59-d874-4447-926f-05e294b8eb9c" />

The suggestions for SocialSpark

Based on the entered keyword, the app presents a recommendation (e.g., "Share something funny or interesting").

<img width="318" height="37" alt="sugg button" src="https://github.com/user-attachments/assets/8be134fa-1069-4a01-ac32-0ee807ee7fc8" />

The Reset Button

For clearing the input and presented recommendation, a reset button is provided.

<img width="323" height="51" alt="Reset" src="https://github.com/user-attachments/assets/f427f5cf-65a0-4302-b86d-521c7de861e1" />

The Error Handling

The app displays a helpful error message if the user inputs an invalid or blank term.

What It Does:

The application compares user input to suggested options:

Time of Day      | Spark Suggestion                                      

Morning          | Send a “Good morning” text                             
Mid‑morning      | Thank a colleague                                      
Afternoon        | Share something funny or interesting                   
Snack Time       | Send a “thinking of you” message                       
Dinner           | Call a friend/relative                                 
Night            | Comment on a friend’s social media post
