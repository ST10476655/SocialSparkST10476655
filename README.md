SocialSpark is a Kotlin-based Android app that offers straightforward, time-based social recommendations to assist users in maintaining social connections during busy days.

App Features

Time‑Based Input
Users type in a time-of-day keyword:
- "morning"
- "afternoon"
- "dinner"
- "night"
  
<img width="323" height="75" alt="Suggestion" src="https://github.com/user-attachments/assets/adae3d59-d874-4447-926f-05e294b8eb9c" />

SocialSpark Suggestions

The app displays a suggestion based on the keyword entered (e.g. “Send a good morning text to a family member”).

<img width="318" height="37" alt="sugg button" src="https://github.com/user-attachments/assets/8be134fa-1069-4a01-ac32-0ee807ee7fc8" />

Reset Button

A reset button is included to clear the input and displayed suggestion.

Error Handling

If the user enters an invalid or empty keyword, the app shows a friendly error message.

How It Works

The app matches user input to preferred suggestions:

Time of Day      | Spark Suggestion                                      

Morning          | Send a “Good morning” text                             
Mid‑morning      | Thank a colleague                                      
Afternoon        | Share something funny or interesting                   
Snack Time       | Send a “thinking of you” message                       
Dinner           | Call a friend/relative                                 
Night            | Comment on a friend’s social media post
